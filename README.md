# prompt

발로란트 VCT 2026 경기 분석 데이터셋을 이용한 경기 승부 예측 
프롬프트 기법에 따라 LLM의 VCT 프로 경기 맵 승패 예측 정확도가 달라지는가?

I. 실험 설계

항목  내용
예측 단위  맵 1개 = 시나리오 1개
목표 샘플 수  100개 맵 (821개 중 선별)
비교 기법 / Few-shot / CoT / Role prompting
사용 모델  Gemini 1.5 Pro
종속변인  예측 정확도 (%)
독립변인  프롬프트 기법

입력 변수
[기본]
- 맵 이름
- 팀A / 팀B 이름
- 양 팀 평균 ACS, KD, ADR, 헤드샷%
- 피스톨 라운드 승패

[에이전트 구성]
- 양 팀 에이전트 픽 목록

[이코노미]
- 풀바이 라운드 수 / 승률
- 에코 라운드 수 / 승률

[킬 패턴]
- 팀별 멀티킬(2K/3K) 횟수
- 팀별 First Kill 수
